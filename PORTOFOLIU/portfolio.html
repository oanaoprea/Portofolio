Oracle Database with SQL

During the second semester of my second year of collage I did a project using Oracle Application and SQL language. The task was to build the database of an online auction company. During the development of the project I have used 10 tables between which I have established connections based on primary and foreign keys and in wchich I have inserted elements. I have also set some constraints regarding the null values, the uniqueness of some elements and the range in which some recordings must fit.
